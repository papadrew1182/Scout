# Family Chore + Routine System (Hearth + Greenlight)

Purpose: clear ownership (not “helping”), age-appropriate responsibility, predictable execution, and reduced mental load on parents.

Kids: **Sadie (13)**, **Townes (10)**, **River (8)**  
Tools: **Hearth** (routines + task visibility) and **Greenlight** (allowance + rewards)

---

## Operating Rules (non-negotiables)

1) **One owner per task.** If it’s shared, it’s orphaned.  
2) **Finishable lists.** Routines should be doable in ~15–25 minutes per block.  
3) **“Done” is defined.** No vague tasks.  
4) **Quiet enforcement.** The checklist + deadline is the boss, not a parent mood.  
5) **One reminder max.** After that: “Check Hearth.”

---

## Standards of Done (paste into Hearth task notes)

### Room Reset = DONE when
- Floor mostly clear (no loose clothes/toys)
- Clothes in hamper
- Dishes/cups removed
- Trash in trash can
- Bed made (simple is fine)

### Common Area Closeout = DONE when
- Cups/plates/trash moved to kitchen
- Blankets folded to basket
- Toys/items put in bins
- Remotes returned
- Floor clear enough a robot vacuum could run

### Trash = DONE when
- Kitchen can checked; if ~75% full → taken out
- Bag tied, taken to outdoor bin
- New bag installed

## Greenlight Allowance: Earned, predictable, low-drama

### Baseline weekly allowance
- Sadie: **$12/week**
- Townes: **$9/week**
- River: **$7/week**

### What earns the baseline
A **Daily Win** = all completed by deadline:
- Morning routine
- After School routine (includes dog-walk item for the right kid)
- Evening routine
- Their ownership chore (Mon–Fri)
- If assigned that day: **Rotating Common Area Closeout**

### Weekly payout (Mon–Fri = 5 days)
- 5 Daily Wins → **100%**
- 4 Daily Wins → **80%**
- 3 Daily Wins → **60%**
- 2 or fewer → **0%** (extras still allowed)

### Extras (optional paid jobs)
Keep a fixed “Extras Menu” with preset prices to avoid negotiation.

---

## School Rewards (paid once per grading period)

Suggested one-time payouts:
- A/B average or Honor Roll: Sadie **$40**, Townes **$30**, River **$25**
- Straight A’s: Sadie **$75**, Townes **$60**, River **$50**
- “No missing assignments” bonus: Sadie **$25**, Townes **$20**, River **$15**

---

## Enforcement (no yelling)

Parent script:
- Reminder (once): “Check Hearth. Get it closed out.”
- If not done by deadline: **screens pause** until completed.
- If not done that day: completed **before weekend fun**.

That’s the whole system: predictable, fair, and enforceable without escalation.
